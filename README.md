# grid-star-wars

## Make the clone to your machine
```bash
git clone https://github.com/jrbigmon/grid-star-wars.git
```
## Install all dependencies
```bash
npm install
```
## Run the project as dev
```bash
npm run dev
```
## Run the project as viewer
```bash
npm run preview
```
## Access route to you view project
http://127.0.0.1:4173/
